# Millennium Technologies
--------------------------

## Purpose

  - Mobile storefront where customers can browse the catalog and purchase products of interest.

## Features

  - Shopping Cart and Wishlist Functionality for anonymous and registered users.
  - Products added to the Shopping Cart can be purchased after a user has registered and logged-in.
  - Logged-in users can give product reviews.

## Technologies Used

1. iOS 15 - IOS Version
2. XCode 13 - IDE
3. Swift 5 - Programming Language
4. UIKit/Storyboard & SwiftUI
5. SQLite Database
6. Apple Pay
